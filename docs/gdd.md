# Game Design Document

## High Concept
3D co-op base defense game where players defend a base and run gate missions for resources.

---

## Core Gameplay
- defend base
- run gates
- gather resources
- survive raids

---

## Core Loop
Prepare → Gate → Extract → Upgrade → Raid

---

## Gate Runs
- instanced
- scaling enemies
- extraction

---

## Base
- central objective
- defenses

---

## Progression
- player upgrades
- base upgrades

---

## Multiplayer
- co-op
- host authoritative

---

## Design Goals
- intense defense
- meaningful decisions
- replayability
