# World Generation System Specification

## Status
Deferred / Backlog

## Purpose
Procedural world generation is not required for the current vertical slice.

The MVP should use one small, readable gate arena. Hand-authored or simple static layout is acceptable and preferred if it helps iterate on fun faster.

## Backlog Direction
World generation may return later for:
- arena variants
- broader gate maps
- POIs
- caves
- exploration routes

## MVP Rule
Do not build procedural exploration maps until the arena survival loop is fun.
