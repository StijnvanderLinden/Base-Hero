# Raid System Specification

## Status
Deferred / Backlog

## Purpose
Major raid systems are not part of the current vertical slice.

The MVP uses escalating gate-arena waves against a central base/core. That pressure is enough to validate active defense, turret upgrades, scrap spending, and essence progression.

## Backlog Direction
Raids may return later as larger base-defense events after the core gate loop is fun.

## MVP Rule
Do not implement town-hall raid checkpoints, raid tech tiers, multi-phase raids, or raid-specific progression until the vertical slice proves the one-more-run loop.
