# Pylon Modifier System Specification

## Status
Deferred / Backlog

## Purpose
Pylon modifiers are not part of the current vertical slice.

The MVP should validate the basic arena survival loop before adding special objective modifiers or repeated-clear challenge layers.

## MVP Rule
Do not implement pylon modifiers, pylon-specific debuffs, repeated pylon clears, or pylon reward scaling until the core loop is fun.
