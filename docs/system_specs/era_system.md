# Era System Specification

## Status
Mostly Deferred For MVP

## Purpose
The Era System can organize future themed content, but the current vertical slice only needs one Stone Age-flavored arena.

## MVP Rule
Do not build multiple eras, era selection complexity, era-specific material trees, or broad era progression until the Stone Age arena loop is fun.

## Current Use
Stone Age may provide:
- visual theme
- first enemy names
- first turret and wall style
- first simple hub unlock names

Anything beyond that is backlog.
