# Core Research Specification

## Status
Simplified For MVP

## Purpose
Core research is currently a simple hub unlock layer paid with essence.

It should not become a deep research tree before the core loop is fun.

## MVP Unlocks
Essence may unlock:
- turret types
- turret upgrade branches
- base capacity
- optional basic player weapon upgrades

## Deferred
Do not implement crystal-gated branches, material-specific research, large augment trees, or special-material requirements for the vertical slice.
