# Pylon System Specification

## Status
Deferred / Backlog

## Purpose
The Pylon System is not part of the current vertical slice.

The MVP gate run is a small arena centered on a base/core, enemy waves, scrap, turret upgrades, milestones, and essence rewards.

## Backlog Direction
Pylons may return later as:
- special gate objectives
- alternate channel events
- map control anchors
- advanced risk-versus-reward structures

## MVP Rule
Do not implement pylon placement, pylon channeling, pylon upgrades, pylon modifiers, or pylon-linked exploration until the core loop is fun.
