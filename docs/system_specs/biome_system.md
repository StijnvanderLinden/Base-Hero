# Biome System Specification

## Status
Deferred / Backlog

## Purpose
Biome systems are not part of the current vertical slice.

Stone Age can provide a simple visual theme for the first arena, but the MVP should not build multiple biomes, biome mechanics, biome enemy families, or biome-specific resource systems.

## MVP Rule
Use only enough Stone Age presentation to make the first arena readable. Defer deeper biome identity until the core loop is fun.
