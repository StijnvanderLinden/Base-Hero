# Resource Props

## Gameplay Purpose

Resource props communicate rewards, pickups, crafting ingredients, or interactable objects. They must be distinct without adding visual clutter.

## Visual Identity

- Simple and readable
- Material-specific silhouettes
- Slightly toy-like
- Clear pickup or storage purpose

## Shape Language

- Strong recognizable outline
- Few large pieces
- Distinct material colors
- Optional blue glow for special resources

## Required Components

Depends on prop type:

- Main resource body
- Container or binding
- Ground contact
- Optional interaction highlight
- Optional value/rarity accent

## Do

- Make each resource type visually distinct.
- Keep props low-poly and simple.
- Use color and silhouette together.
- Keep pickup scale readable but not huge.
- Put origin at ground center or visual center depending on use.

## Don't

- Make resources look like enemies or traps.
- Use tiny scattered pieces that clutter the arena.
- Add realistic texture noise.
- Create too many similar brown/gray props.

## Variation Ideas

- Scrap bundle
- Stone chunks
- Wood stack
- Metal crate
- Energy crystal
- Reward chest
- Upgrade crate

## Example Roles

- Scrap pickup
- Essence container
- Build material bundle
- Reward chest
- Special unlock prop
