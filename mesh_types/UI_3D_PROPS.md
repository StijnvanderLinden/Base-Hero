# UI 3D Props

## Gameplay Purpose

UI 3D props are in-world objects that communicate interaction, status, upgrade choices, or objective information without relying only on screen UI.

## Visual Identity

- Simple, readable, and symbolic
- Built into the world
- Engineer-tech fantasy
- Clear interactive affordance

## Shape Language

- Icon-like silhouettes
- Clean panels, pedestals, levers, buttons, signs, cores
- Strong contrast for interactable areas
- Minimal detail

## Required Components

- Main readable body
- Interaction surface or focal point
- Optional blue energy accent
- Stable base or mount
- Clear facing direction if used like a console

## Do

- Make interaction points obvious.
- Keep text/icon surfaces large enough to read.
- Use blue glow for active or usable states.
- Use simple shapes that do not compete with combat objects.
- Put origin at base center or interaction pivot.

## Don't

- Make UI props overly decorative.
- Use tiny text as the only communication.
- Make them look like turrets, traps, or enemies.
- Add high-poly screens or complex panels.
- Use futuristic hologram styling unless explicitly requested.

## Variation Ideas

- Upgrade pedestal
- Gate console
- Base core indicator
- Resource deposit marker
- Turret upgrade marker
- Locked upgrade sign
- In-world arrow marker

## Example Roles

- Hub unlock station
- Gate start console
- Turret upgrade prompt prop
- Base health indicator
- Objective marker
